# EgoHAnG Project Page

A static academic project page inspired by the SITAR project-page layout.

## Files
- `index.html` — main page
- `style.css` — page styling
- `assets/egohang_architecture.png` — architecture figure extracted from the paper
- `assets/qualitative_results.png` — qualitative-results figure extracted from the paper
- `EgoHAnG_ICPR26.pdf` — paper PDF

## Before publishing
1. Replace each author `href="#"` in `index.html` with the correct homepage/profile URL.
2. If an official ICPR 2026 paper page becomes available, link the venue text.
3. Verify the GitHub repository URL and repository visibility.
4. Deploy the folder with GitHub Pages.

## GitHub Pages
Push all files in this folder to a repository, then enable Pages from the repository settings.
